# GEDI - Lab 3 Reproduction

Lightweight, fully reproducible clustering reproduction package for Part B of the lab.

## Quick start

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

Run notebooks in order from the project root:
1. `notebooks/01_main_reproduction.ipynb`
2. `notebooks/02_ablation_study.ipynb`
3. `notebooks/03_new_dataset_evaluation.ipynb`

> See [notes.txt](notes.txt) for full descriptions, run instructions, and important notes.
